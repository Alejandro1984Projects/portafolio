PROYECTO TIENDA GENÉRICA

PROYECTO DE SOFTWARE PARA GESTIONAR TRANSACCIONES COMERCIALES DE UNA TIENDA GENÉRICA.

Descripción

El propósito de este proyecto es un software escrito en lenguaje Java para Web para gestionar las transacciones comerciales de una tienda que sea funcional para cualquier tipo de comercio.

Detalles

Manejo de proveedores, clientes, compras, ventas, y productos.

validación por nombre de usuario y contraseña.

Gestion de reportes.

Informacion adicional

  •  Cliente: Etapa Practica Universidad El Bosque
  •  Lenguaje: Java y JavaScript
  •  Fecha: Diciembre 2021
  •  Ubicación: Bogota
